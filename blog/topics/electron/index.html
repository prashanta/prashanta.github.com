Trying out Electron - a timer menu bar app
There is a growing list of applications made from Github's Electron. Electron basically take cares of low level OS related engagements and provides a cozy framework to create multi-platform native applications using web technologies - HTML, Javascript and CSS. So its easy to understand why programmers are eagerly adapting this cool tool for developing native apps.
